## 1.0.0

Initial release as `colorpicker`.

- Added a full-featured Flutter color picker with hex, HSV, alpha, gradient, preset, and recent-color workflows.
- Added design-tool style inline controls including `ColorPickerRow`, `ColorPickerTrigger`, `ColorPickerPanel`, and dialog helpers.
- Added unified paint, layer, swatch, and gradient models for state management and serialization.
- Added a Soft SaaS UI styled example app covering picker, layer, gradient, and primitive component demos.
